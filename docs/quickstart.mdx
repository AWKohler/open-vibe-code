## 1. Create a project

Sign in at [botflow.io](https://botflow.io) and click **New Project**. Choose a template — React, Next.js, or a blank Node.js starter.

## 2. Edit your code

The workspace opens with a file tree on the left, an editor in the center, and a live preview on the right. Every save re-runs your app instantly — no manual refresh needed.

## 3. Use the AI assistant

Open the AI panel and describe what you want to build. The assistant can create files, modify existing code, and explain what it did.

```
"Add a dark mode toggle to the navbar"
```

## 4. Deploy

When you're ready, click **Deploy** in the toolbar. Your app is published to a unique URL within seconds.
