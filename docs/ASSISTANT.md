# Botflow Documentation Assistant

Botflow is a browser-based IDE that lets you build, run, and deploy full-stack web applications without installing anything locally. It uses WebContainers to run a real Node.js environment directly in the browser tab.

## Key concepts

- **Projects** — workspaces that contain your code, files, and settings
- **Editor** — Monaco-based code editor with syntax highlighting and autocomplete
- **AI Assistant** — generates and edits code using OpenAI and Anthropic models
- **Preview** — live in-browser preview of your running app
- **Deploy** — one-click publishing to a public URL

## Stack

Botflow supports web templates (Vite, Next.js) out of the box.
